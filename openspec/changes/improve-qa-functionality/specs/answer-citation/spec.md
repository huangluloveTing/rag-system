## ADDED Requirements

### Requirement: 答案引用来源标注

系统 SHALL 在生成答案时自动标注引用来源，支持用户点击跳转查看原文。

#### Scenario: 成功标注单个引用
- **WHEN** LLM 生成答案并引用了知识库内容
- **THEN** 答案中在引用句末显示 `[1]` 格式的上标链接

#### Scenario: 成功标注多个引用
- **WHEN** 答案引用了多个文档片段
- **THEN** 每个引用位置分别标注 `[1]`、`[2]`、`[3]` 等，按出现顺序编号

#### Scenario: 点击引用跳转
- **WHEN** 用户点击引用标记 `[1]`
- **THEN** 展开侧边面板显示来源文档信息（文件名、页码、chunk 内容）

#### Scenario: 引用悬停预览
- **WHEN** 用户鼠标悬停在引用标记上
- **THEN** 显示 Tooltip 包含来源文件名和相似度分数

#### Scenario: 无引用时的处理
- **WHEN** 答案未使用知识库检索（LLM 内置知识回答）
- **THEN** 答案后显示 "基于通用知识生成，未引用特定文档" 提示
